Help

● Installation

To Install MaineCoon head on over to the Download Page and download the
appropriate package for your computer. Next run the installer,
this is pretty straightforward and

No errors should occurr during this.
If errors come about make sure to tell me about it
by contacting me.

After you run the
installer you should be greeted
with a wizard to install, simply hit
install and wait for MaineCoon to install

You can go grab a cup of coffee during this or go show affection to your faviorite feline.

After it says "Installation Finished" MaineCoon Will be setup, you will find a shortcut
on your desktop or you can search for it in the windows
menu by pressing the windows key and searching.

● General Usage

So one you've gotten it installed and launched you should
be greeted with herbert the mainecoon and a nice
user interface for MaineCoon.

Running it immediatly is pretty easy all you have to do
is input the number of parts you want to create be careful
as trying to create to many parts will more than
likely crash your game before it crashes
the server . I would not advise more than 500K parts
and about 100K ish scripts will work on most medium server
but use it how you want. And after that you need to attach it to
roblox by hitting attach, then you can just go and
execute it by hitting execute.

Now if you want to be more efficient and effective, you might want to use
some other options, Humanoid Parts and Nest Parts,
are a great way of doing this to big servers
though this will be more cpu intensive for you
so make sure you have enough machine for Mainecoon
And the other options do exactly as they say so.

Each server is going to be a bit different to crash than other so
you will need to tweak you settings.

You can also save your current setting to a file by hitting
the "save" button.



Remember if you need any other help you can contact
me by going to the contact page.
