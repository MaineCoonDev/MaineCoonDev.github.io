MaineCoon Server Crasher.

Mainecoon Server crasher is a powerful roblox server crasher that is soon to be
open source. It is fast, reliable and borderline unpatchable.

How does it work?

MaineCoon works by heaps and heaps of objects and bloat scripts
and injecting them into pre-existing server objects, so much so that at first this will
cause mild to extreme server lag. After a while the server will no longer be able to keep up
with the amount of scripts objects and requests that the server just sort of falls in on it's self.
