MaineCoon Level 7 Executor.

Mainecoon is a powerful Lua Script Executor that is soon to be
open source. It is fast, reliable and borderline unpatchable.

How does it work?

MaineCoon works by quitely injecting itself into roblox
and disguising itself as a part of roblox. Mainecoon also is able to execute high level scripts with simple backend payloads to preform not only
local changes to scripts but server wide changes. Mainecoon is currently in it's alpha stages so bugs are expected
so please make sure to make us aware of it!
